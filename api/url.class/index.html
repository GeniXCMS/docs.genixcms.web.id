    Url Class API Reference¶

    The Url class is the central routing and path generation engine in GeniXCMS. it handles the creation of all system links, ensuring they comply with either standard query-string formats or "Smart URL" (friendly permalink) formats.

    🛠️ Static Methods¶

    Url::post(int|string $id): string¶

    Generates a full URL for a specific post.

    • Hook: Applies the post_url filter.
    • Smart URL: https://site.com/post-slug/
    • Standard: https://site.com/?post=123

    Url::page(string $slug): string¶

    Generates a full URL for a static page.

    • Smart URL: https://site.com/page-slug/
    • Standard: https://site.com/?page=page-slug

    Url::cat(int $id): string¶

    Generates a URL for a category archive.

    • Smart URL: https://site.com/category/12/category-name/

    Url::ajax(string $resource, string|array $action = '', array $params = []): string¶

    Generates a secure internal Ajax endpoint URL.

    • $resource: The Ajax class/resource to target.
    • $action: Specific action to perform (or array of params).
    • Security: Automatically includes the system TOKEN.
    • Smart URL: https://site.com/ajax/resource/TOKEN/?action=save

    Url::api(string $resource, string $id = '', array $params = []): string¶

    Generates a GeniXCMS RESTful API endpoint URL.

    • $resource: API resource name (e.g., 'posts').
    • $id: Specific resource identifier/ID.
    • Smart URL: https://site.com/api/v1/posts/123

    Url::mod(string $name, string $act = '', array $params = []): string¶

    Generates a URL for a module backend.

    • Smart URL: https://site.com/mod/module-name/?act=edit&id=1

    Url::thumb(string $src, string $type = '', string $size = '', string $align = ''): string¶

    Generates a dynamic thumbnail path using the ThumbFly engine.

    • Parameters:
      • type: crop or resize.
      • size: e.g., 400x300.
      • align: e.g., center.
    • Note: Strips duplicate thumb prefixes and automatically uses Site::$cdn.

    Url::theme(): string¶

    Returns the base URL path of the currently active theme (e.g., inc/themes/my-theme/).

    Url::custom(string $url): string¶

    Returns a custom URL as provided. Simple passthrough utility.

    Url::sitemap(string $section = null): string¶

    Generates a URL for the sitemap (XML).

    • Smart URL: https://site.com/sitemap/posts.xml
    • Standard: https://site.com/index.php?page=sitemap

    Url::rss(): string¶

    Generates the URL for the main RSS feed.

    • Smart URL: https://site.com/rss/
    • Standard: https://site.com/?rss

    Url::tag(string $name): string¶

    Generates a URL for a specific tag archive.

    • Smart URL: https://site.com/tag/my-tag/
    • Standard: https://site.com/?tag=my-tag

    Url::user(string $username, string $section = ''): string¶

    Generates a URL for a frontend user profile page.

    • $section: Optional profile section (e.g., 'settings', 'orders').
    • Smart URL: https://site.com/user/johndoe/settings/
    • Standard: https://site.com/?user=johndoe&section=settings

    Url::author(string $username, string $type = ''): string¶

    Generates a URL for an author's public profile/archive.

    • Smart URL: https://site.com/author/johndoe/

    Url::search(): string¶

    Generates the URL for the search results page.

    Url::slug(int $id): string¶

    Internal utility to retrieve the URL slug for a specific post ID from the database.

    🧭 Navigation & Utility Methods¶

    Url::current(): string¶

    Returns the complete URL of the page currently being viewed (including protocol, host, and request URI).

    Url::breadcrumbs(array $data = []): string¶

    Generates standardized Bootstrap/Tailwind-ready HTML breadcrumbs.

    • Hook: Applies the breadcrumbs_filter.

    Url::archive(int $month, int $year): string¶

    Generates a URL for a specific time-based archive.

    Url::flag(string $lang): string¶

    Generates a language switcher link for the given language code.

    Url::login(string $params = ''): string / Url::logout(): string¶

    Generates URLs for the system authentication pages.

    ⚡ Smart URL logic¶

    The behavior of this class is globally controlled by the SMART_URL constant (defined in gxadmin/inc/config.php or Options).

    • If permalink_use_index_php is enabled, all Smart URLs will automatically include index.php/ in the path for servers without URL rewriting support.
    • If multilang_enable is active, the class automatically injects the active language prefix (e.g., /en/) into the generated paths.
